================================================================================
SparseGPT 改进版本 - 测试套件使用说明
================================================================================

重要提示: 所有测试脚本已配置为仅使用 GPU 2号和3号 (CUDA_VISIBLE_DEVICES=2,3)
          如需修改，请编辑脚本开头的 export CUDA_VISIBLE_DEVICES=2,3

一、快速开始（推荐）
────────────────────────────────────────────────────────────────────────────────
  运行快速验证（15分钟）:
    ./quick_test.sh
  
  查看结果:
    cat quick_test_results/quick_results.csv


二、完整验证
────────────────────────────────────────────────────────────────────────────────
  一键运行所有测试（1-5小时）:
    ./run_full_validation.sh
  
  查看完整报告:
    cat analysis_output/ANALYSIS_REPORT.md
  
  查看可视化图表:
    ls analysis_output/*.png


三、测试脚本说明
────────────────────────────────────────────────────────────────────────────────
  quick_test.sh              - 快速验证 (15分钟)
  statistical_test.sh        - 统计显著性测试 (1小时)
  benchmark_complexity.sh    - 计算复杂度对比 (20分钟)
  comprehensive_test.sh      - 全面配置测试 (4小时)
  run_full_validation.sh     - 一键完整验证 (1-5小时)
  
  analyze_results.py         - 结果分析工具（自动生成报告）


四、判断标准
────────────────────────────────────────────────────────────────────────────────
  改进有效 ✓:
    - WikiText2 PPL 改进 > 2%
    - T-test p-value < 0.05
    - 时间开销 < 1.5x
  
  改进有限 ⚠:
    - WikiText2 PPL 改进 0-2%
    - 需要优化参数
  
  改进无效 ✗:
    - WikiText2 PPL 未改进
    - 需要重新设计


五、结果目录
────────────────────────────────────────────────────────────────────────────────
  quick_test_results/        - 快速测试结果
  statistical_results/       - 统计测试结果
  complexity_results/        - 复杂度测试结果
  comprehensive_results/     - 全面测试结果
  analysis_output/           - 分析报告和图表
    └── ANALYSIS_REPORT.md   - 最终报告（重点查看）


六、常用命令
────────────────────────────────────────────────────────────────────────────────
  # 快速验证
  ./quick_test.sh
  cat quick_test_results/quick_results.csv
  
  # 完整验证
  ./run_full_validation.sh
  cat analysis_output/ANALYSIS_REPORT.md
  
  # 重新生成分析报告
  python3 analyze_results.py


七、建议流程
────────────────────────────────────────────────────────────────────────────────
  1. 首次验证: ./quick_test.sh (确认改进有效)
  2. 详细验证: ./run_full_validation.sh (获取完整数据)
  3. 查看报告: cat analysis_output/ANALYSIS_REPORT.md
  4. 使用数据: 撰写论文/技术报告
